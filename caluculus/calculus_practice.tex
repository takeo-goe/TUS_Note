% ===== 例題1 =====
\begin{itembox}[l]{例題1}
$x,y,z$平面および下の図の平面に囲まれた領域の体積を求めよ。

\vspace{10pt}

\begin{center}
\includegraphics[width=0.5\linewidth]{images/number1.png}
\end{center}

\end{itembox}



\begin{center}
\includegraphics[width=0.8\linewidth]{images/number2.png}
\end{center}

\vspace{10pt}
\fbox{方針}\par



①平面を$z = f(x,y)$の形式で明示する.\par

②積分範囲を$D = \{(x,y)|xとyの条件\}$という形式で明示する.\par

③二重積分を立式する.

\vspace{10pt}
\fbox{解答}\par
平面は
\begin{align*}
    z = 1-x-y
\end{align*}
とかける. 指定の領域の体積を計算するには, 積分範囲を
\begin{align*}
    D = \{(x,y)|0 \le x \le 1, 0 \le y \le 1-x\}
\end{align*}
として二重積分を行う.

\begin{align*}
    V &= \iint_{D}^{} z dx dy\\
      &= \int_{0}^{1} (\int_{0}^{1-x}(1-x-y)dy)dx  \\
      &=...  
\end{align*}



% === 例題2 ===
\begin{itembox}[l]{例題2}
\begin{align*}
    \iint_{D}^{} (3x + 2y) dx dy, D = \{(x,y) | 0 \le x \le 1, 0 \le y \le 2\}
\end{align*}
を累次積分で求めよ.
\end{itembox}

\fbox{解答}
\begin{align*}
    V &= \int_{0}^{2} (\int_{0}^{1} (3x + 2y)dx)dy\\
      &= \int_{0}^{2} \left[\frac{3}{2} x^2 + 2yx \right]_{0}^{1}\\
      &= \int_{0}^{2} (\frac{3}{2} + 2y)dy\\
      &= \left[\frac{3}{2}y + y^2 \right]_{0}^{2}\\
      &= 7.
\end{align*}





% === 例題3 ===
\begin{itembox}[l]{例題3}
\begin{align*}
    V = \iint_{D}^{} x y^2 dx dy, D = \{(x,y) | 0 \le x \le 1, 0 \le y \le 2\}
\end{align*}
を変数分離によって求めよ.
\end{itembox}

\fbox{解答}
\begin{align*}
    V &= \int_{0}^{1} x\, dx \int_{0}^{2} y^2\, dy\\
      &= \left[ \frac{1}{2} x^2 \right]_{0}^{1}
         \left[ \frac{1}{3} y^3 \right]_{0}^{2}\\
      &= \frac{4}{3}.
\end{align*}

% === 例題5 ===
\begin{itembox}[l]{例題5}
以下の積分を極座標への変換を用いて計算せよ.
\begin{align*}
    I = \iint_{D}^{} \sqrt{x^2 + y^2} dx dy, D = \left\{(x,y) | x^2 + y^2 \le 1, x \ge 0\right\}
\end{align*}
\end{itembox}

\fbox{解答}\par
$x = r \cos \theta, y = r \sin \theta$という関係式を用いて変数変換を行う.

\vspace{130pt}

①積分範囲の変換\par

積分範囲$D$の条件はそれぞれ, 
\begin{align*}
    &x^2 + y^2 \le 1\\
    &\rightarrow r^2(\cos \theta^2 + \sin \theta^2) \le 1\\
    &\rightarrow -1 \le r \le 1 \rightarrow 0 \le r \le 1 (\text{距離の非負性})
\end{align*}

また,

\begin{align*}
    x &\ge 0\\
    &\rightarrow r \cos \theta \ge 0\\
    &\rightarrow \cos \theta \ge 0\\
    &\rightarrow -\frac{\pi}{2} \le \theta \le \frac{\pi}{2}
\end{align*}

②被積分関数の変換\par

\begin{align*}
    \sqrt{x^2 + y^2} = \sqrt{r^2(\cos^2 \theta + \sin^2 \theta)} = r
\end{align*}

③面積要素の変換\par

\begin{align*}
    dx dy = rdr d\theta
\end{align*}

以上より

\begin{align*}
    I &= \iint_{D'}^{} f(x(r,\theta),y(r,\theta)) rdrd\theta\\
      &= \int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \int_{0}^{1} r^2 dr d\theta = \frac{\pi}{3}
\end{align*}
